Initialization

The included init.php loads seeds.php, which is used to set up the database and populate it from incldued files. Click here to visit this page and initialize the database.

Extra credit features included
  • A user can stored a secret question/answer (accessible via Settings page). If present, these are presented during the "Forgot Password" flow before a user receives the reset password via email
  • When browsing course details, an advisor can use links to the student records to view each student
Some other cool things implemented
  • Course and user details pages are fully bookmarkable (done via a $_GET request based on ID rather than stored in $_SESSION)
  • Notes store session_id of the advising session, so that the two are associated (visible in the "Advising Sessions" tab of the user details page)
  • Sidebar on the user details page includes some extra details, including a gravatar image that will be visible if the user's email has a gravatar associated with it
